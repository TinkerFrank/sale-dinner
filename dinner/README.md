# Sale Dinner Suggest

A web app that suggests dinners based on grocery sale items, with **audible step-by-step cooking instructions** powered by ElevenLabs.

## Quick start (Python — recommended)

```powershell
cd dinner
py server.py
```

On Windows, use `py` if `python` is not recognized. You can also double-click `start.bat`.

Open **http://localhost:3000** in your browser.

## Alternative (Node.js)

```powershell
cd dinner
npm install
npm start
```

## Setup

1. Copy `.env.example` to `.env` if needed.
2. Add your API keys:

```
ELEVENLABS_API_KEY=your_key_here
APIFY_TOKEN=your_apify_token_here
```

Both keys are only used on the server — they are never sent to the browser.

## How it works

1. Click **Load sales from Apify** — pulls deals from your pre-scraped dataset (Jumbo aanbiedingen).
2. Or paste sale items manually (one per line).
3. Click **Find dinner ideas** — recipes are scored by how many sale ingredients they use.
4. Pick a recipe and use **Listen** or **Play all steps** for audible cooking guidance.

Configure your dataset in `.env`:

```
APIFY_TOKEN=your_token
APIFY_DATASET_ID=T9VGfs4oRGnhhlcPW
OPENAI_API_KEY=your_openai_key
OPENAI_MODEL=gpt-4o-mini
```

With `OPENAI_API_KEY` set, recipes are generated by GPT from your sale items. Without it, a rule-based fallback is used.

## Security note

Never commit `.env` or share your API key publicly. If a key is exposed, rotate it in your ElevenLabs dashboard.
